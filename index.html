☁️ 开启全员云端共享

默认数据存在你自己的浏览器（本地模式）。想让所有人看到同一份、谁都能加记录，按下面步骤接入免费的 Firebase 数据库：

  1. 打开 console.firebase.google.com，用谷歌账号登录，点「创建项目」（名字随意，关掉Google Analytics）
  2. 项目建好后，左侧菜单进「构建 → Realtime Database」，点「创建数据库」，地区选默认，规则选「以测试模式启动」
  3. 点项目首页的 </> 图标（添加Web应用），起个名注册，会得到一段 firebaseConfig = {...} 配置
  4. 把那段 { ... } 大括号里的内容整段粘贴到下面（包含 apiKey、databaseURL 等）：

⚠️ 配置保存在你的浏览器里。要让朋友也连上同一个数据库，把这段配置发给他们，让他们也在自己的「云端设置」里粘一次即可。
